# Banking Transaction Analysis using Python & Tableau

## Project Overview
This project analyzes banking transaction data to identify financial trends,
withdrawal patterns, deposit activity, and balance behavior.

## Tools Used
- Python
- Pandas
- Google Colab
- Tableau Public

## Features
- Data Cleaning
- Exploratory Data Analysis
- Financial Trend Visualization
- Interactive Dashboard

## Key Insights
- Withdrawal activity varied across months
- Deposits remained relatively stable
- Balance trend showed overall consistency
- High-value transactions were identified
